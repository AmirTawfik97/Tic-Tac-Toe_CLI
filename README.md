TicTacToe

A Simple Tic Tac Toe game running on command line

The game asks for 2 player names

Then asks for maximum scor (between 1 & 10)

The game randomly generate the first player

The first player is always assigned to "X"

The second player is then assigned to "O"

Whatever the round outcome (Win or Draw), players switch symbols

Players select rom 1 - 9 to input their symbols

If the cell is not empty players will retry for another cell

Rounds will keep going till either players score the maximum score
